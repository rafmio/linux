/proc/PID/fd
Тут содержится по одной сивольной ссылке для каждого fd, открытого процессом. Кажд. из этих символьн.ссылок имеет название, совпадающ. с номером дескр-ра, н-р, /proc/1968/df/1 явл.симв.ссылкой на stdout процесса 1968.
	Для удобства, любой процесс может обратиться к своему собс.каталогу с помощ.симв.ссылки /proc/self

/proc/PID/task
Т.к. некот.атрибуты для потоков в группе потоков различяются, с Linux 2.4 добавился каталк task.
Для кажд. в пр-се потока ядров предоставляет подкаталог /proc/PID/task/TID
TID - идентификатор потока (Thread ID)
